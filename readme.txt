【labyrinth】
　・Unityのプロジェクト。C:\Users\Public\Documents\Unity Projects\直下に置けばプロジェクトが開けるはず。

【Unity_Game】
　・ゲームのexeファイル。OSに合わせてWin_x86\Win_x86.exeかWin_x86_64\Win_x86_64.exeを選ぶ。
　・矢印キーで操作。ドラゴンレーダーを頼りに時間内に太一くんを捕まえればユニティちゃんの勝ち。

　　↑：画面奥方向に進む
　　↓：方向転換
　　→：右を向く
　　←：左を向く
　
　・デバッグ用機能
　　1:カメラ切り替え
　　移動中にspace：ジャンプ
    移動中にZ：勝利のポーズ
    移動中にL：負けのポーズ
    移動中にC：小さいジャンプ
　　X：その場でジャンプ
